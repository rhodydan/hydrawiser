 How to Get a Google Maps API Key

        
      

      

        
Google Maps services require a Cloud Platform API Key with billing enabled. Follow these steps to set one up for free (Google offers $200 free monthly credits, which easily covers personal use):

        
        
    
          
  1. 
            Open the Google Cloud Credentials Console:
            
    
            Go to Google Cloud Credentials Console . Select or create a project if prompted.
          
    2. 
          
  2. 
            Create an API Key:
            
    
            Click + Create Credentials at the top, select API Key, and copy the generated key.
          
    3. 
          
  3. 
            Enable Required Libraries:
            
    
            To use all features of this application (map displays, drawing tools, address autocomplete, and high-res Static Map rendering), you must enable these 3 APIs in the Google API Library :
            
      
              
    • Maps JavaScript API  (interactive canvas and drawing tools)
      • 
              
    • Places API  (address search bar autocomplete)
      • 
              
    • Maps Static API  (high-resolution composite rendering engine)
      • 
            
    
          
    4. 
          
  4. 
            Enable Billing:
            
    
            Google requires active billing to allow loading map tiles. Make sure a billing account is linked to your project in the Billing Console .
          
    5. 
        

        
        

          
          

            Important Security Tip: If hosting this page publicly, restrict the API key in the Credentials Console. Under "API restrictions", select "Restrict key" and select only the three APIs mentioned above.
